# Гайд по функции Enemies Counter (Счетчик противников)

## Общее описание
Функция Enemies Counter позволяет отображать количество противников на экране игры Standoff 2. Информация отображается в верхней части экрана по центру с черным фоном за числом.

## Файлы реализации
1. `src/standoff/enemies_counter.h` - основная логика подсчета и отображения
2. `src/standoff/menu.h` - интеграция в меню и сохранение настроек
3. `src/main.cpp` - вызов функции отображения

## Основные функции

### countEnemies()
- Сканирует список игроков в игре
- Определяет противников (игроки с другой команды с HP > 0)
- Возвращает количество противников

### renderEnemiesCounter()
- Вызывается каждый кадр при активации опции
- Подсчитывает количество противников
- Отображает текст "Enemies: [число]" в верхней части экрана по центру
- Добавляет черный фон за числом

## Интеграция в меню
- Вкладка ESP -> чекбокс "Enemies" для включения/выключения функции
- Состояние сохраняется в конфигурационный файл

## Сохранение настроек
- Параметр enemies_counter_enabled сохраняется и загружается из config.cfg
- Функции SaveConfig() и LoadConfig() в menu.h обрабатывают этот параметр

## Особенности реализации
- Использует RPM (read process memory) для получения данных из процесса игры
- Работает с адресами памяти игры Standoff 2
- Использует ImGui для отображения на экране
- Позиционирование текста по центру верхней части экрана
